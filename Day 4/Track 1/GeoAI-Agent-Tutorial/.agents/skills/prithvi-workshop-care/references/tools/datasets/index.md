# datasets tools

Auxiliary dataset queries used to help interpret vague user requests (e.g., infer likely hazard type/date window). These do not expand inference capabilities.

## Files
- `query_active_fires.md` — FIRMS active fire signals in bbox/date-range.
- `query_surface_water.md` — OPERA DSWx surface water signals in bbox/date-range.
- `query_fire_history.md` — MTBS historical fire perimeters intersecting bbox/date-range.
- `query_crop_landcover.md` — CDL crop/landcover summary for bbox (and optionally year).
